## Phase 1: ❤️ Accessibility & Contrast (Day 1)
*تأثير مباشر على الوصولية والقراءة — أخطر النقط*

### 1.1 — WCAG Contrast Fixes 🔴
| Current | Issue | Fix |
|---------|-------|-----|
| `#6b6390` ← `#a098c0` | Hijri date on `#0a0e27` — fails AA at 12px | Lighter lavender |
| `#3a3450` ← `#6b6380` | Verse/footer — nearly invisible | Medium purple-gray |
| `#5a5470` ← `#7a7390` | Placeholder — too dark | Slightly brighter |
| `opacity: 0.3` ← `0.4` | Past prayer rows — readability low | Bump opacity |

**Files:** `src/App.css`

### 1.2 — `prefers-reduced-motion` 🟡
Wrap ALL animations in `@media (prefers-reduced-motion: reduce)`:
- `bgShift`, `fadeUp`, `timerBgShift`, `particleFloat`, `ornamentPulse`, `fadeIn`
- Default: `animation: none !important; transition: none !important;`

**Files:** `src/App.css`

### 1.3 — `::selection` + Focus Indicators 🟡
```css
::selection { background: rgba(212, 168, 67, 0.3); color: #e8e0d0; }
*:focus-visible { outline: 2px solid #d4a843; outline-offset: 2px; }
```

**Files:** `src/App.css`

*تأثير على الموثوقية*

If the app can't fetch prayer times (no network, API down):

- Show "آخر تحديث: [time]" note

### 5.2 — Responsive for Ultrawide/Small Screens 🟢
- 1366×768 (your laptop): already tested ✅

- 1024×768: ensure scroll works without clipping
