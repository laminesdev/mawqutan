```ts
const CACHE_KEY = 'mawqutan-prayers';
const CACHE_TIME_KEY = 'mawqutan-prayers-time';

export function getCachedPrayers(): PrayerInfo[] | null {
  try {
    const raw = localStorage.getItem(CACHE_KEY);
    const time = localStorage.getItem(CACHE_TIME_KEY);
    if (!raw || !time) return null;
    const age = Date.now() - parseInt(time);
    if (age > 86400000) return null; // stale after 1 day
    return JSON.parse(raw);
  } catch { return null; }
}

export function setCachedPrayers(prayers: PrayerInfo[]) {
  localStorage.setItem(CACHE_KEY, JSON.stringify(prayers));
  localStorage.setItem(CACHE_TIME_KEY, String(Date.now()));
}
```

Islamic prayer time app for desktop. Dark theme with gold accent. Bilingual Arabic/Latin interface.

## Colors
- Background: `var(--color-bg)` / `#0a0e27`
- Surface: `var(--color-surface)` / `rgba(232, 224, 208, 0.05)`
- Primary text: `var(--color-text-primary)` / `#e8e0d0`
- Secondary text: `var(--color-text-secondary)` / `#8b83a0`
- Accent: `var(--color-accent)` / `#d4a843`

## Typography
- Arabic titling/body: Noto Naskh Arabic (serif)
- Latin & numerals: Inter (sans-serif)

## Components
- RegionSelect: centered card, max 360px, input + select + cta
- PrayerTimes: fullscreen vertical stack, header + next card + list + footer
- Timer: overlay with SVG progress ring

## Do's
- Use gold for active/current/next elements only
- Keep opacity-based hierarchy (past prayers fade)
- Prefer text hierarchy over boxes/colors

## Don'ts
- Don't add more accent colors
- Don't use glassmorphism
- Don't place interactive elements in drag regions
